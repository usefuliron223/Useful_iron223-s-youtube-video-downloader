# Useful_iron223-s-youtube-video-downloader 
a fully free youtube video downloader
paste link to download, can choose available resolution and fps
Prerequisites

This app requires the following Python libraries:

    CustomTkinter: For the modern user interface.

    yt-dlp: To handle video extraction and downloading.

    Pillow: To process the application icon.

    FFmpeg: Required on your system for merging high-quality video and audio.
